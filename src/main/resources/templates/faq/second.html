Q2: 비밀번호를 잊어버렸을 때 어떻게 하나요?

비밀번호를 잊어버린 경우, 다음과 같은 방법으로 비밀번호를 재설정할 수 있습니다:

  1. 로그인 페이지에서 비밀번호 찾기 버튼을 클릭합니다.
  2. 등록된 이메일 주소나 휴대폰 번호를 입력합니다.
  3. 입력한 이메일로 비밀번호 재설정 링크가 발송됩니다. 해당 링크를 클릭하여 새 비밀번호를 설정합니다.
  4. 새 비밀번호를 입력하고 비밀번호 변경 완료를 클릭하면 새로운 비밀번호로 로그인할 수 있습니다.

재설정 링크가 24시간 이내에 만료되므로, 빠르게 비밀번호를 변경해주세요.
